# Weather Diary

**Автор:** Аракелов Тимур

## Описание программы

Weather Diary — это графическое приложение для ведения дневника погоды. Позволяет:
- Добавлять записи (дата, температура, описание, осадки)
- Просматривать все записи в таблице
- Фильтровать записи по дате или по температуре
- Сохранять данные в JSON-файл (автоматически при добавлении)
- Загружать данные при запуске

## Как запустить приложение

1. Убедитесь, что установлен Python 3.6 или выше.
2. Скачайте или склонируйте репозиторий:

```bash
git clone https://github.com/parklir/Weather-Diary-
cd main.py
